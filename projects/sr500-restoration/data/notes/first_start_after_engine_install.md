# Yamaha SR500 – første oppstart etter montering av motor

Kort, praktisk sjekkliste for første oppstart etter at motoren er satt tilbake i ramma.

## 1. Før du prøver å starte

### Motor og innfesting
- [ ] Alle motorbolter montert og trukket til
- [ ] Avstandsstykker / spacere på riktig plass
- [ ] Ingen kabler, slanger eller ledninger klemt mellom motor og ramme

### Oljesystem
- [ ] Alle oljelinjer montert riktig vei
- [ ] Banjobolter riktig plassert
- [ ] Nye eller gode tetningsskiver montert
- [ ] Oljelinjene ligger uten knekk, vridning eller strekk
- [ ] Ingen åpne oljeporter igjen
- [ ] Oljefilter montert
- [ ] Oljetank / ramme fylt med korrekt oljemengde

### Drivstoff / innsug
- [ ] Tank montert eller ren midlertidig drivstofftilførsel koblet opp
- [ ] Bensinkran fungerer
- [ ] Forgasser koblet riktig
- [ ] Innsugsgummi tett og uten sprekker
- [ ] Luftfilter / innsugsløsning montert

### Tenningssystem / elektrisk
- [ ] Jordforbindelser montert
- [ ] CDI, coil og plugghette koblet riktig
- [ ] Tennplugg riktig type og gap
- [ ] Batteri tilkoblet hvis brukt i oppsettet
- [ ] Kill switch i riktig stilling

### Mekanisk grunnsjekk
- [ ] Ventilklaring kontrollert
- [ ] Kamkjede / ventilmekanisme virker normal
- [ ] Motoren går rundt for hånd / med kick uten unormal låsing
- [ ] Girkasse i nøytral
- [ ] Eksos montert

## 2. Første priming før ekte oppstart
- [ ] Tenningssystem av hvis du vil prime rolig først
- [ ] Bruk dekompresjon og kick motoren gjennom flere rolige slag
- [ ] Kjenn etter at motoren går jevnt rundt mekanisk
- [ ] Se etter umiddelbare oljelekkasjer rundt banjoer og filterområde

## 3. Første oppstart
- [ ] Tenningssystem på
- [ ] Choke etter behov
- [ ] Start motoren uten å ruse den
- [ ] La den gå rolig på tomgang
- [ ] Ikke gi mye gass de første sekundene

## 4. Det viktigste: sjekk oljeretur
- [ ] Åpne peile-/påfyllingslokket i ramma forsiktig
- [ ] Se ned i oljetanken med lys
- [ ] Bekreft at du ser retur / bevegelse i oljen
- [ ] Litt pulserende retur er normalt på en stor singel

### Hvis du IKKE ser retur
- [ ] Stopp motoren
- [ ] Sjekk linjeføring, banjobolter, filter og oljemengde
- [ ] Ikke fortsett å la motoren gå “for å se om det kommer seg”

## 5. Under første minutt
- [ ] Sjekk lekkasjer ved alle oljekoblinger
- [ ] Lytt etter tørr, skarp ventil-/kamlyd som ikke høres normal ut
- [ ] Bekreft stabil tomgang
- [ ] Bekreft at eksos og innsug oppfører seg normalt

## 6. Etter kort varmgang
- [ ] Stopp motoren
- [ ] Se over alle oljeledninger og koblinger på nytt
- [ ] Sjekk etter svetting rundt banjoer og filterhus
- [ ] Kontroller oljenivå etter korrekt prosedyre når systemet har sirkulert
- [ ] Ettertrekk bare der det faktisk trengs og med fornuft

## 7. Rød-flagg – stopp med en gang hvis
- [ ] Ingen synlig oljeretur
- [ ] Kraftig lekkasje
- [ ] Unormal metallisk lyd
- [ ] Motoren blir fort varm og “tørr” i lyden
- [ ] Banjo / oljelinje svetter eller spruter olje

## Kortversjon – minimumsrutine
1. Fyll olje
2. Dobbeltsjekk alle oljelinjer
3. Kick gjennom rolig først
4. Start på tomgang
5. Bekreft oljeretur i ramma
6. Stopp og sjekk for lekkasje

## Praktisk SR500-notat
På SR500 kan oljenivået være misvisende hvis sykkelen har stått lenge og olje har siget ned i motoren. Bruk derfor ikke bare peilepinnen som sannhet før du har bekreftet faktisk sirkulasjon / retur.
