- Satt opp Vegvesen AKF enkeltoppslag i workspace.
- Script: `vegvesen_lookup.py`
- Kortkommando: `regnr`
- Bruker endpoint `GET /enkeltoppslag/kjoretoydata`
- Auth må sendes som header `SVV-Authorization: Apikey <VEGVESEN_API_KEY>`
- `.env` i workspace inneholder `VEGVESEN_API_KEY`
- Bruk:
  - `./regnr NU1537`
  - `./vegvesen_lookup.py NU1537`
  - `./vegvesen_lookup.py --understellsnummer <VIN>`
- Når Johnny ber om å slå opp et kjennemerke, kan dette scriptet brukes direkte.
- Struktur for lagrede Vegvesen-oppslag: `bibliotek/vegvesen/<kjoretoytype>/<merke>/<modell>/<kjennemerke>.json`
- Første strukturerte lagring: `bibliotek/vegvesen/motorsykkel/yamaha/sr500-2j4/BB5092.json`
- Indeksfil for lagrede kjøretøy: `bibliotek/vegvesen/index.json`
- `vegvesen_lookup.py` lagrer nå automatisk oppslag i riktig mappe og oppdaterer indeksfilen, med `--no-save` som unntak.
- Preferanse: ved forespørsler om kjennemerke/understellsnummer skal biblioteket sjekkes først; bruk Vegvesen-oppslag bare hvis data ikke allerede finnes lagret.
