# Yamaha SR500 – tekniske spesifikasjoner (så komplett som mulig)

Dette notatet er satt sammen med vekt på **1980–1981 fabrikkdata** fra SR500 80–81 service-/spec-sider, supplert med noen sekundærkilder der fabrikkdata ikke var like lett tilgjengelig.

## Modell / grunninfo
- Modellfamilie: **Yamaha SR500**
- Typisk modellkode: **2J2**
- Motor: gateversjon av Yamaha sin klassiske store ensylinder-plattform
- Startsystem: **kickstart**

## Motor
- Type: **luftkjølt, 4-takt, ensylindret, SOHC**
- Ventiler: **2 per sylinder**
- Slagvolum: **499 cc**
- Boring x slag: **87.0 x 84.0 mm**
- **Kompresjonsrate: 9.0:1**
- Forbrenningskammer: **hemisfærisk**
- Toppakning tykkelse: **1.0 mm**
- Smøresystem: **dry sump / tørrsumpsystem**
- Oljepumpe: **trochoidal pump**
- Tenning: **CDI**

## Effekt / ytelse
- Effekt: **ca. 27 hk ved 6000 rpm**
- Toppfart: **ca. 135 km/t**
- Moment: enkelte sekundærkilder oppgir rundt **36 Nm ved 5500 rpm**

## Innsug / drivstoff
- Forgasser: **Mikuni VM34SS**
- Forgasser-ID: **2J200**
- Main jet: **#300**
- Air jet: **#80**
- Needle jet: **P-8**
- Cutaway: **3.5**
- Pilot jet: **#25**
- Starter jet: **#50**
- Float height: **23.5 ± 1 mm**
- Tomgang: **1100 rpm**
- Drivstofftype: **regular gasoline / vanlig bensin**
- Tankvolum: **12.0 liter**

## Tennings- og elsystem
- Systemspenning: **12 V**
- Batteri: **12N7-3B / G.S.**
- Batterikapasitet: **12V, 7Ah**
- Laderate: **0.7 A x 10 timer**
- Spesifikk vekt (elektrolytt): **1.280**
- Regulatorspenning: **14.5 ± 0.5 V**

## Tennplugg
- Fabrikkdata fra 80–81-specside: **NGK BP6ES** eller **ND W20EP**
- Tennplugg-gap: **0.7–0.8 mm**
- Lokal delekatalog bekrefter også **NGK BP6ES**

## Oljekapasitet
- Total mengde: **2.4 liter**
- Periodisk oljeskift: **2.0 liter**
- Ved filterskift: **2.1 liter**
- Anbefalt olje i kilden: **Yamalube 20W/40** eller tilsvarende

## Drivverk
- Girkasse: **5-trinns**
- Sluttdrift: **kjede**
- Kjedetype: **DID 50 HDSS**
- Antall ledd: **103L + joint** (fabrikkdata fra specsiden)
- Kjededimensjon / moderne referanse: **530**
- Framdrev: **16 tenner**
- Bakdrev: **42 tenner**

## Mål og vekt
- Lengde: **2105 mm**
- Bredde: **835 mm**
- Høyde: **1150 mm**
- Akselavstand: **1410 mm**
- Bakkeklaring: **165 mm**
- Nettovekt / tørrvekt: **160–163 kg**
  - kilder for 80–81 oppgir både **160 kg** og **163 kg**
- Kjøreklar / wet weight: **ca. 174 kg**
- Setehøyde: **ca. 810 mm**
  - dette er en vanlig publisert SR500-spec, men var ikke tydelig i 80–81 fabrikkutdragene jeg fikk tak i

## Styring / geometri
- Gaffelvinkel / caster: **27.5°**
- Trail: **117 mm**
- Lock-to-lock vinkel: **41.5°**

## Forhjulsoppheng
- Type: **teleskopgaffel**
- Fjæringsvei foran: **150 mm**
- Inner tube diameter: **35 mm**
- Gaffelolje mengde: **182 cc per gaffelbein**
- Gaffelolje type: **Yamaha fork oil 10 wt. eller tilsvarende**

## Bakhjulsoppheng
- Type: **svingarm med coil spring / oil damper**
- Støtdempervandring: **80 mm**
- Bakhjulsvandring: **110 mm**
- Bakfjær fri lengde: **216.5 mm**
- Bakfjær set length (soft position): **198 mm**

## Bremser
- Foran: **hydraulisk skivebrems**
- Skivedimensjon foran: **298 x 5 mm**
- Slitasjegrense skive: **4.5 mm**
- Beleggtykkelse: **11 mm**
- Min. beleggtykkelse: **6 mm**
- Hovedsylinder diameter: **14.0 mm**
- Kalipersylinder diameter: **38.18 mm**
- Bremsevæske: **DOT 3**

- Bak: **trommelbrems (leading trailing)**
- Trommeldiameter: **150 mm**
- Bånd / lining thickness: **4 mm**
- Slitasjegrense bak: **2 mm**

## Hjul / dekk
- Fordekk: **3.50-S19-4PR**
- Bakdekk: **4.00-S18-4PR**
- Felg foran: **1.85-19 aluminium**
- Felg bak: **2.15-18 aluminium**

## Servicepunkter som er nyttige å ha med
- Ventilklaring kald, innsug: **0.10 mm**
- Ventilklaring kald, eksos: **0.15 mm**

## Hva som er mest pålitelig her
Høyest tillit:
- boring x slag
- kompresjonsrate
- oljemengder
- batteri
- tennplugg/gap
- akselavstand
- bakkeklaring
- bremser
- dekk/felgdimensjoner
- gaffeldata

Litt mindre sikkert / markedsavhengig:
- setehøyde
- eksakt tørrvekt (160 vs 163 kg)
- momenttall fra sekundærkilder
