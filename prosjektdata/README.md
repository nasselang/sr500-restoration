# SR500 Restaurering — Prosjektdata

Dette inneholder alle datafiler for Yamaha SR500-restaureringsprosjektet (BB5092).

## Innhold

- `data/` — CSV-filer med delenumre, bilder av deler, notater, handleliste
- `memory/` — daglige logger og prosjektminne
- `reference/` — delelister og verkstedmanual
- `bibliotek/` — registreringsdata og vognkort

Prosjektet drives via OpenClaw + SR500-agenten.
